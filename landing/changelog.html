Changelog

All notable changes to Claudit

v0.5.0 December 2025
  • Feature MCP Server Editing — Edit global MCP server configurations directly within Claudit
  • Feature Nested Folder Commands — Display commands from nested folders in the commands view
  • Improvement Project-Level Views — Updated MCP servers and commands view on projects level
  • Fix Layout issues on agents page
  • Fix Projects commands folder now opens correctly
  • Improvement License changed to MIT
v0.4.1 December 2025
  • Feature Enhanced Tray Menu — Burn rate, session info, primary model, recent projects, quick actions, and notifications toggle
  • Improvement Tray Performance — Menu interactions now instant with cached data
  • Fix Settings sync between tray and UI now works bidirectionally
v0.4.0 December 2025
  • Feature Session Browser — Browse and view all session conversations for each project
  • Feature In-Conversation Search — Search within session conversations with Cmd+F, text highlighting, and match navigation
  • Feature Session Export — Export sessions to HTML format
  • Feature Session Resume — Copy resume command or open terminal with session resume
  • Improvement Centralized Pricing — All model pricing in single source of truth, visible in Settings
  • Fix Session cost calculation now includes cache tokens
  • Fix Project names with dashes now display correctly
v0.3.0 December 2025
  • Feature Projects Page — Browse all Claude Code projects with usage stats, costs, and last used dates
  • Feature AI Suggestions — Get Claude-powered suggestions for custom agents, commands, CLAUDE.md improvements
  • Feature Project Details — View CLAUDE.md content, project commands, and MCP servers
  • Feature Agents Page — Browse global and project-specific Claude agents with content preview
  • Feature Plugins Page — Browse installed Claude plugins with marketplace info
  • Feature Sidebar Navigation — New collapsible sidebar with all app sections
  • Fix AI suggestions now display immediately after generation
v0.2.0 December 2025
  • Feature Close button (X) in header to minimize to tray
  • Feature Hide from dock on macOS when window is closed
  • Fix App icon now displays correctly (orange C logo) in favicon and headers
  • Improvement Cross-platform messaging updated (no longer claims macOS-only)
v0.1.0 December 2025
  • Feature Initial release with real-time Claude Code usage monitoring
  • Feature Menu bar integration with live token consumption display
  • Feature Analytics dashboard with interactive charts for tokens, costs, and models
  • Feature Smart notifications via Claude Code hooks integration
  • Feature Per-project and per-model usage breakdown
  • Feature Burn rate tracking (tokens/min and $/hour)
  • Feature Session-based usage tracking (5-hour blocks)
  • Feature Configurable tray menu display options
