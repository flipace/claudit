Changelog

All notable changes to Claudit

v0.2.0 December 2025
  • Feature Close button (X) in header to minimize to tray
  • Feature Hide from dock on macOS when window is closed
  • Fix App icon now displays correctly (orange C logo) in favicon and headers
  • Improvement Cross-platform messaging updated (no longer claims macOS-only)
v0.1.0 December 2025
  • Feature Initial release with real-time Claude Code usage monitoring
  • Feature Menu bar integration with live token consumption display
  • Feature Analytics dashboard with interactive charts for tokens, costs, and models
  • Feature Smart notifications via Claude Code hooks integration
  • Feature Per-project and per-model usage breakdown
  • Feature Burn rate tracking (tokens/min and $/hour)
  • Feature Session-based usage tracking (5-hour blocks)
  • Feature Configurable tray menu display options
